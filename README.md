# Laravel / Semantic UI
A basic Laravel application implementing Semantic UI.

## Installation

```bash
git clone https://github.com/ctessier/laravel-semantic-ui
cd laravel-semantic-ui
composer install
yarn install
bower install
cp .env.sample .env
php artisan key:generate
php artisan migrate
```
